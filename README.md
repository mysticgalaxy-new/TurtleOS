TurtleOS is a light weight Kernel designed for maximum performance.

## Everything

- `src/boot.s`
- `src/linker.ld`
- `src/kernel.c`
- `src/console.c`
- `src/vga.c`
- `src/serial.c`
- `src/keyboard.c`
- `grub/grub.cfg`
- `Makefile`

## How to run
run:
```bash
make clean
make all
make run
```
